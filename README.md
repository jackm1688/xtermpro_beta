# xtermpro

**xtermpro** 是基于 **Qt 6** 的跨平台桌面工作台：在统一界面中整合 **SSH 交互终端**、**本地 Shell**、**SFTP**、**远程桌面（RDP，开发测试中）**、会话树与保存的连接、隧道与代理、AI 辅助与脚本工具等能力。目标平台为 **Windows、macOS、Linux**，并强调三端视觉与交互一致。

> 架构与模块边界详见 [`docs/XTERMPRO_ARCHITECTURE_AND_RULES.md`](docs/XTERMPRO_ARCHITECTURE_AND_RULES.md)。

---

## 功能概览

### 终端与会话

| 能力 | 说明 |
|------|------|
| **SSH 终端** | 基于 libssh 的交互式 Shell；支持密码、私钥等认证方式；占位标签页 → 连接成功后再挂载真实终端，避免多会话并发时界面错乱。 |
| **本地终端** | 本机 Shell（PTY / Windows ConPTY–winpty 等与 `xqtermw` 集成），与 SSH 标签同属中间工作区管理。 |
| **会话列表** | 左侧会话树：保存的连接（通常为 `.ini` 配置）、搜索、上下文菜单打开 SSH / SFTP 等。 |
| **多标签工作区** | `TabWindowWidget` 统一管理各类标签：连接中、已连接、断开、错误态及重连所需的主机、认证、代理、主题、编码、字号等元数据。 |

### SFTP

| 能力 | 说明 |
|------|------|
| **SFTP 命令终端** | 本地 `sftp>` 风格命令解释器，底层为 SFTP API（非远程 Shell 透传）；支持 `put` / `get` 等异步传输与取消。 |
| **SFTP 图形浏览** | 图形化 SFTP 浏览标签（与 SFTP 终端并列存在于产品中）。 |

**说明：** 若会话启用 **代理跳转等代理模式**，产品侧会限制从会话树直接打开 SFTP（避免未经设计的代理 + SFTP 组合）。

### 远程桌面（RDP）— 开发测试中

**状态说明：** RDP 目前处于 **开发测试** 阶段，能力 **尚不完善**，可能存在连接稳定性、兼容性或体验上的问题，不适合作为生产唯一远程桌面方案。

实现上通过 **FreeRDP** 相关适配层提供 **RDP** 连接与桌面标签（连接 / 断开 / 错误等状态与 SSH 标签类似管理）。网关与配置可参考 [`docs/RDP_GATEWAY_CONFIGURATION_GUIDE.md`](docs/RDP_GATEWAY_CONFIGURATION_GUIDE.md) 等文档。

### 隧道、代理与同步

| 能力 | 说明 |
|------|------|
| **SSH 隧道** | 本地转发、远程转发、动态转发（SOCKS）等隧道配置的创建与管理。 |
| **SSH 代理（Jump 等）** | 代理主机、认证等与连接编排相关的配置（连接对话框与隧道/代理管理界面联动）。 |
| **SSH 命令代理** | 独立于会话的代理配置（如 SOCKS5、`-L` / `-R` 类转发），用于命令行工具等场景。 |
| **云凭证与 COS 同步** | 菜单中的云 AK 与 **SSH/COS 同步** 能力（对象存储同步相关实现见 `docs/SSH_COS_SYNC_*.md` 等系列文档）。 |

### AI 辅助面板

- 支持多个 **HTTP API 提供商**（如 DeepSeek、Qwen、ChatGPT 兼容接口、百炼等），配置保存在 `ai_secret_config/*.ini`。
- **SSH 会话提供商**：不向公网发对话请求，而是解析并下发命令，例如：
  - `@当前会话 <命令>`
  - `@全部会话 <命令>`
  - `@<会话名> <命令>`

### 安全与运维工具

| 能力 | 说明 |
|------|------|
| **凭据 / 密钥管理** | 保存与管理连接凭据、密钥文件等（仍须遵循本地安全实践）。 |
| **命令拦截（CmdFilter）** | 按规则拦截危险或不合规命令，在终端提交前生效。 |
| **会话操作日志** | 记录会话相关操作，便于审计与排查。 |
| **网卡与流量** | 网卡信息、网络流量与会话维度的网络监控相关工具。 |

### 其它

| 能力 | 说明 |
|------|------|
| **终端录制** | 对终端内容进行视频录制（开始/停止由菜单控制）。 |
| **脚本库** | 脚本的管理与执行入口（详见 `docs/SCRIPT_LIBRARY_*.md`）。 |
| **工具站点 / 内置页** | 工具类站点管理、不可随意关闭的固定工具标签等。 |
| **界面语言** | 支持自动 / 中文 / 英文等 UI 语言切换（具体条目见「配置」菜单）。 |
| **激活** | 授权与激活管理（见 `docs/ACTIVATION_CODE_GUIDE.md`）。 |

### 终端引擎（xqtermw）

`src/xqtermw` 提供终端渲染、仿真、会话、**Windows ConPTY / winpty**、搜索、历史与键盘映射等；日常功能开发应优先在 `uimain` / `appcore` 层完成，仅在确属仿真或 PTY 问题时再改动该子树。

## 界面布局（主窗口）

- **左侧**：`SessionListWidget` — 会话树与入口操作。  
- **中间**：`TabWindowWidget` — 运行时标签（SSH / 本地 / SFTP / RDP（开发测试中）/ 文件等）。  
- **右侧**：AI 辅助面板（可隐藏）。  

顶层菜单负责新建连接、本地 Shell、打开文件、视图开关、各类配置对话框以及录制与帮助等。

---

## 系统要求（概要）

- **Windows**：Windows 10 或更高；Qt 6、CMake 3.16+、Visual Studio 或 MinGW（以你方 CI/脚本为准）。  
- **macOS**：较新 macOS；Xcode 或 Command Line Tools；Qt 6；CMake 3.16+。  
- **Linux**：现代发行版；Qt 6；GCC/Clang；CMake 3.16+。
