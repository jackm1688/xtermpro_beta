功能迭代

新增应用级 UI 主题切换能力，包含外观设置入口、主题状态同步、共享 UiTheme 扩展，以及对应 PRD/SRD/任务文档。
新增工具站点标签页可见性设置，并补齐工具站点菜单英文翻译。
新增 SFTP 并行传输队列，包含服务层队列调度、浏览器 UI 队列展示、传输状态本地化，以及 PRD/SRD/任务文档。
SFTP 文件列表新增“文件类型”列，并完善文件类型显示与翻译策略。
支持显式构建版本号，调整 CMake、Info.plist、macOS 构建脚本及相关展示逻辑。
补充 macOS 构建脚本使用文档。
优化改进

大规模扩展统一 UI 主题覆盖范围，涉及 AI 助手、连接页、RDP、SFTP、Kubernetes、密钥/凭据/云 AK 管理、脚本库、命令过滤、API Key、工具站点管理等页面。
统一上下文菜单、提示弹窗、覆盖层弹窗、表格选中态、SFTP 弹窗、Kubernetes 详情页/监控页等视觉风格。
Kubernetes 工作台完成多轮布局优化：
适配窗口宽度；
优化资源详情区域；
恢复详情侧栏；
减少底部留白；
调整搜索框和上下文控件布局；
优化监控工具栏；
降低工具栏布局抖动。
主窗口与 Kubernetes 工作台窗口行为优化：
启动居中；
Kubernetes 工作台居中、最大化、保持在屏幕范围内；
减少 splitter 拖拽闪烁。
会话侧栏更紧凑，session tree 间距收窄，并支持悬停显示完整会话名称。
工作区标签栏保持可见，避免被隐藏导致导航不明确。
云同步界面标题命名调整。
